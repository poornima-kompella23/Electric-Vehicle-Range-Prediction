# Electric-Vehicle-Range-Prediction
Electric Vehicle Range Prediction using Python and Machine Learning. This project analyzes vehicle parameters such as battery capacity, speed, terrain, and temperature to estimate the driving range of an electric vehicle. It uses data processing and machine learning techniques to improve prediction accuracy and help optimize EV performance.
# Dataset features
- Battery capacity(kwh)
- Vehicle weigth(kg)
- Average Speed(km/hr)
- Temperature
# Technologies Used
- Python
- Pandas
- Scikit-learn
- Mathplotlib
# Project Files
- Electric_vechicle_range.ipynb : Machine learning model Code
- ev_dataset.csv: Dataset used for training and testing
# output
The model predicts the estimated driving range of an electric vehicle based on given input parameters
